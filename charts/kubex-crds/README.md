# kubex-crds

This chart installs the CustomResourceDefinitions (CRDs) required for Kubex Automation Engine.

The `kubex-crds` chart and the Kubex Automation Engine chart are available for evaluation and early adoption. For production-grade automation workloads, customers should continue using the Kubex Automation Controller chart.

## Usage
Install this chart before installing or upgrading the Kubex Automation Engine chart.
